Virtbase is a relatively new hosting provider offering affordable, reliable hardware. This GitHub repository focuses on building a native iOS & macOS application as a companion to the web dashboard, allowing you to manage your servers, firewall, and backups more conveniently.

> [!NOTE]
> Development is scheduled to start on Wednesday, 28 January. Until then, feel free to join the Discord server. Please note that pull requests related to the currently non-existent codebase will be closed. Pull requests related to the README, license, or other documentation will be reviewed as usual.

> [!IMPORTANT]
> The Virtbase app for iOS & macOS is currently under development and should not be considered stable. Bugs and errors are expected.
> If you find a bug, have suggestions, or want to propose new features, feel free to contact [@abgespeichert](https://discord.com/users/1417576672823083098) on Discord, or create a new ticket on the official [Virtbase Discord](https://discord.gg/ywrqTubzh5).
